# Run Locally

Clone the repository:

```git clone https://github.com/YOUR_USERNAME/notnotion.git```

Navigate to the project folder:

```cd notnotion```

Install dependencies:

```npm install```

Start the development server:

```npm run dev```

Open in your browser at http://localhost:5173
